## 📖 Overview

The **Beverage Sales Dashboard** is an interactive Microsoft Excel dashboard designed to analyze beverage sales performance across different **retailers, beverage brands, delivery companies, cities, and regions**.

This dashboard provides a quick overview of key business metrics using KPIs, Pivot Tables, Pivot Charts, and Slicers, helping users make data-driven decisions with ease.
